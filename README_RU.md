# Инструкция по использованию

## Требования
- Power BI Desktop (бесплатная программа от Microsoft)
- Windows 10/11 (рекомендуется)

## Установка Power BI Desktop

### Windows:
1. Скачайте с https://powerbi.microsoft.com/desktop/
2. Запустите установщик
3. Следуйте инструкциям установки

## Открытие дашборда

1. Запустите Power BI Desktop
2. File → Open → Выберите `Formula1-Dashboard.pbix`
3. Дождитесь загрузки (может занять 10-30 секунд)

## Как пользоваться

### Навигация
- Используйте кнопки на главной странице для перехода между разделами
- Или используйте Page Navigator (навигатор страниц) слева

### Фильтрация данных
- **Слайсеры** - интерактивные фильтры по бокам страниц
- Выберите год, команду, гонщика или трассу для фильтрации

### Детальный анализ (Drill-through)
1. Кликните правой кнопкой на элемент данных (например, имя гонщика)
2. Выберите "Drill through" → нужная страница
3. Откроется детальная аналитика по выбранному элементу

## Основные страницы

### 1. Formula 1 (Главная)
Обзор ключевых метрик и навигация

### 2. Drivers (Гонщики)
- Статистика гонщиков
- Сравнение производительности
- Историческая статистика

### 3. Constructors (Команды)
- Рейтинг команд
- Очки и достижения
- Динамика по сезонам

### 4. Races & Circuits (Гонки и трассы)
- Результаты гонок
- Карта трасс
- Статистика по трассам

## Экспорт данных

### Экспорт в Excel:
1. Кликните на визуализацию (график/таблицу)
2. Три точки (⋯) в правом верхнем углу
3. "Export data"
4. Выберите формат и сохраните

### Экспорт в PDF:
1. File → Export → Export to PDF
2. Выберите страницы
3. Сохраните

## Обновление данных

Если источники данных доступны:
1. Home → Refresh
2. Или настройте автообновление в Gateway

*Примечание: В этом файле данные статичны (декабрь 2024)*

## Возможные проблемы

### Файл не открывается
**Решение:** Обновите Power BI Desktop до последней версии

### Медленная загрузка
**Решение:** 
- Закройте другие программы
- Файл большой (9.4 MB), загрузка может занять время

### Ошибка подключения к данным
**Решение:** Это нормально для портфолио проекта. Данные встроены в файл.

## Системные требования

- **ОС:** Windows 10/11
- **RAM:** Минимум 4 GB (рекомендуется 8 GB)
- **Место на диске:** 3-5 GB для Power BI Desktop
